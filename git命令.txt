1.设置用户信息
    git config --global user.name <username>
    git config --global user.email <mailaddress>
2.初始化本地仓库
    git init
3.将工作区文件添加到暂存区
    git add
    git add .  add所有文件
4.查看文件状态
    git status
5.提交到本地仓库
    git commit  -m '备注信息'
6.查看本地仓库的历史版本信息
    git log
7.从暂存区找回删除的文件
    git checkout 文件名
8.从git管理中永久删除文件
    git rm 文件名
9.版本回退
    git reset --hard 版本号
10.查看整个回溯版本的历史信息
    git reflog
11.在.gitignore配置不被追踪的文件
12.添加远程仓库地址
    git remote add 远程仓库别名 远程仓库的http地址
13.查看远程仓库
    git remote
14.拉取别人的仓库
    git clone
15.拉取自己的仓库------必须保持远程仓库和本地仓库的一致性
    git pull --rebase
    在进行git push
16.查看分支
    git branch
17.新建分支
    git branch 分支名
18.切换分支
    git checkout 分支名
19.分支合并(主分支marster上合并其他分支)
    git merge 需要合并的分支名
